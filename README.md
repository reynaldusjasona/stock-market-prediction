# Stock Market Prediction Web App
> FYP-26-S2-26 | University of Wollongong at SIM

A web-based stock market forecasting and recommendation 
platform focused on the US market, combining technical 
analysis tools with AI-powered price prediction and 
personalized investment recommendations — designed to 
provide a TradingView-like experience with predictive 
and personalized capabilities.

## Core Features
- User Registration & Login
- Real-time Stock Data Integration
- AI-Powered Price Prediction Engine
- Interactive Data Visualization & Charts
- Technical Indicators 
- Simplified Order Book
- Personalized Stock Recommendations
- Alert & Event Detection
- News & Sentiment Analysis
- Top Gainers & Losers + Stock Screener
